Обновление аналитики до v8.3: редизайн графиков и новые отчеты

Версия v8.3 включает следующие ключевые изменения:

1. Визуальное обновление:
- Пиктограмма "Трактор" заменена на "Плавильная печь" (🔥).
- Добавлена легенда на основной график производства.

2. Редизайн графика «Эффективность и Выплавка»:
- Теперь столбики показывают себестоимость в формате "тг/кг".
- Объем выплавленного металла представлен сплошной ярко-зеленой линией для наглядного сравнения.
- Увеличен шрифт и улучшена читаемость подписей данных.

3. Глубокая аналитика (под спойлерами):
- Добавлен блок "🏆 Лидеры и антилидеры" для быстрого поиска самых эффективных смен.
- Добавлен "📊 График-рейтинг", сортирующий смены по цене за 1 кг металла.

4. Логика данных:
- Принудительная сортировка смен: "День" теперь всегда идет перед "Ночью".
- Улучшена функция автозамены устаревших терминов (КЭЗ -> остатки шихтовых).
- Оптимизированы алгоритмы расчета расхода ДТ.
